# Mood Tracker

A web app for tracking daily mood and journaling, built with Firebase for authentication and data storage.

### **What It Does:**
Login/signup with email and password (Firebase Authentication)
Calendar view for picking a day and logging a mood (happy, sad, mad, crying, or neutral), using hand-drawn icons
Journal entry tied to that day's mood
Personalized suggestions on a confirmation page based on the mood logged, plus a summary of that day's entry

### **What I'd Build Next:**
Move mood/journal history into the calendar view itself (e.g. color-coding
past days by mood), rather than only logging forward
Add basic form validation and error states on login/signup
Anonymous auth as an option, so a user can try the app before creating a
real account
